Two Sum II — sorted array

while l < r:
    s = nums[l] + nums[r]
    if s == target:  return [l+1, r+1]
    elif s < target: l += 1  # need bigger
    else:            r -= 1  # need smaller

Container With Most Water

while l < r:
    area = max(area,
        min(height[l], height[r]) * (r - l))
    if height[l] >= height[r]: r -= 1  # move shorter
    else:                       l += 1

Reverse In Place

while l < r:
    arr[l], arr[r] = arr[r], arr[l]
    l += 1; r -= 1

Fast / Slow — Cycle Detection

slow = fast = head
while fast and fast.next:
    slow = slow.next
    fast = fast.next.next
    if slow == fast: return True  # cycle
return False

Remove Duplicates — sorted, in-place

k = 1
for i in range(1, len(nums)):
    if nums[i] != nums[i-1]:
        nums[k] = nums[i]
        k += 1
return k  # k = new length

Window Size Formula

window_size = r - l + 1  # inclusive
# while l < r  → pointers never cross
# while l <= r → pointers can land same

When to Use

# Sorted array + pair/triplet problem
# Fast/slow for linked list cycle/midpoint
# In-place write (slow=write, fast=read)
# O(n) replaces O(n^2) brute force
