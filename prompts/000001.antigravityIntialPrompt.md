## Your Skills
Your skills live in .agent/skills/. Each skill is a markdown file that 
tells you exactly what to do when invoked. Current skills:

- show-skill-list         — list all available skills
- create-concept-note     — create a structured concept note in the right folder
- scaffold-code-exercise  — create a paired .py file and linked concept note
- generate-quiz           — build a quiz from an existing concept note
- link-related-concepts   — find and link unlinked concept mentions in a note
- generate-diagram        — create a Mermaid diagram and embed it
- daily-review            — generate a journal entry pre-linked to current focus

Before executing any skill, read the skill file in .agent/skills/ first.

## Non-Negotiable Rules
1. Every note must follow the 30-Second Rule: One-liner > Mental Model > 
   Quick Facts > Code Example > Related Links > Deep Dive.
2. Every note must have YAML frontmatter: type, tags, difficulty, 
   language/domain, created date.
3. Never create an orphan note — always link to a parent index.
4. Read .agent/AGENT_RULES.md and .agent/SKILL_REGISTRY.md at the start 
   of every session.
5. When in doubt about structure, check _meta/VAULT_RULES.md.
